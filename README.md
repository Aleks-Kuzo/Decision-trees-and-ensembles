# Decision-trees-and-ensembles
В данном репозитории собраны работы по изучению и применению алгоритмов машинного обучения на основе деревьев

# 10_1_ Решающее дерево.ipynb
В этом ноутбуке изучаются основы работы DecisionTreeRegressor на примере данных о стоимости жилья в Калифорнии. Программа демонстрирует процесс обучения модели, визуализирует структуру дерева и наглядно показывает проблему переобучения при неограниченном росте глубины дерева.

# 10_3_Tuning Decision Tree.ipynb
Этот файл посвящен оптимизации деревьев решений. С помощью инструментов кросс-валидации и GridSearchCV в нем реализован автоматизированный поиск лучших гиперпараметров (глубина, минимальное количество объектов в листе), а также рассматривается метод стрижки (прунинга) дерева для улучшения его обобщающей способности.

# 10_4_Дз Решающие деревья.ipynb
Практическая работа по классификации клиентов телеком-оператора для предсказания их оттока. В ноутбуке выполняется полный цикл подготовки данных (обработка категориальных признаков), обучение модели классификации и оценка качества с использованием метрики ROC-AUC и анализа важности признаков.

# 11_1_Разложение ошибки и разброс.ipynb
Теоретико-практический файл, посвященный разложению ошибки на смещение (bias) и разброс (variance). Программа сравнивает линейную регрессию и решающее дерево, демонстрируя, как высокая гибкость деревьев приводит к низкому смещению, но высокому разбросу, что требует дополнительной регуляризации.

# 11_2_Random Forest.ipynb
Работа посвящена изучению Случайного леса. Программа сравнивает работу одиночного дерева и ансамбля деревьев, доказывая устойчивость леса к переобучению и демонстрируя постепенное улучшение качества предсказания при увеличении количества базовых моделей в ансамбле.

# 11_4_Дз Задержка самолетов.ipynb
Финальная работа, направленная на решение задачи бинарной классификации задержек авиарейсов. В файле проводится сравнительный анализ трех мощных алгоритмов: решающего дерева, случайного леса и градиентного бустинга, а также исследуется декомпозиция ошибки на смещение (bias) и разброс (variance).
